# Roadmap

## 0.1

- Project scaffold.
- GTK4/libadwaita shell.
- GPGME contexts.
- Key listing.
- OpenPGP encrypt/decrypt helpers.
- Security docs and AppImage scripts.

## 0.2

- Key creation assistant.
- Import/export keys.
- Signature and verification UI.
- Drag-and-drop file workflows.
- Safer temporary archive handling.

## 0.3

- S/MIME certificate UI.
- Certificate chain viewer.
- CRL/OCSP status display where available.
- WKD/HKPS search with explicit network prompts.

## 0.4

- Secure vault implementation.
- Auto-lock.
- Access log without sensitive data.
- High Security mode UI.

## 0.5

- Smartcard workflows for YubiKey, Nitrokey, and OpenPGP Card.
- AppImage update metadata.
- Cross-distribution AppImage test matrix.

## Experimental

- liboqs-backed Post-Quantum Lab.
- Hybrid OpenPGP plus post-quantum experiments.
- Import/export format for experimental data with explicit warnings.
