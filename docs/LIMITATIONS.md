# Limitations

This initial repository is a secure starter, not a complete production release.

Current limitations:

- No completed key generation assistant.
- No completed revocation certificate workflow.
- No completed S/MIME certificate chain UI.
- No completed secure vault.
- No WKD/HKPS network implementation.
- No smartcard UI.
- No liboqs integration.
- No VM/container AppImage matrix yet.

Security limitation:

The Post-Quantum Lab is not standard OpenPGP and must not be represented as drop-in GnuPG compatibility.
